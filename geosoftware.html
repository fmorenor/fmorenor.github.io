Software de fotogrametría

AU4 — Compilación y edición estereoscópica

Una potente herramienta que permite la creación rápida y ágil de cartografía de alta calidad y precisión. Presente en 5 continentes, AU4 combina la compilación estereoscópica con automatización avanzada y soporte para múltiples sensores.

Compilación estereoscópica

Herramientas de edición ágiles con control de calidad topológico y revisión en tiempo real.

Curvas de nivel en tiempo real

Generación automática con colores, etiquetas e intervalos totalmente personalizables.

Integración LiDAR

Importación de datos LiDAR y visualización de nubes de puntos en estereoscopía.

Múltiples sensores y WMS/WFS

Compatible con cámaras aéreas, imágenes satelitales y servicios web estándar.

Plataforma web geoespacial

eCarto — Colaboración geoespacial en la nube

eCarto es una plataforma de colaboración geoespacial basada en web que permite crear mapas sencillos, conectando backends complejos y todo tipo de datos georreferenciados desde cualquier dispositivo.

A través de la API de eCarto, crea mapas geográficos en tiempo real o aplicaciones y plugins más complejos desde cualquier lugar con conexión a Internet.
